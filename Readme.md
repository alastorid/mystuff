## getprice.coolpc.sh

It can fetch price information from [http://coolpc.com.tw](http://coolpc.com.tw) aka 天x屋.

A website sells 3c products in Taiwan.


### Prerequisite

+ iconv
+ curl

### How to use?

```bash
chmod +x ./getprice.coolpc.sh
./getprice.coolpc.sh
```
